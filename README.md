# 🛒 Drools Analytics Dashboard

An enterprise-level Power BI dashboard designed to analyze sales, customers, inventory, feedback, and marketing performance for a quick-commerce business model.  
The solution enables business stakeholders to monitor KPIs, identify growth opportunities, optimize inventory, and improve customer experience through data-driven insights.

---

# 📊 Project Overview

This Power BI dashboard provides end-to-end business analytics across multiple operational areas including Sales, Customer Insights, Inventory Management, Customer Feedback, and Marketing Performance.

The dashboard was developed using Power BI, DAX, Power Query, and Excel-based datasets to simulate real-world quick-commerce operations similar to Blinkit / Instamart business models.

---

# 🚀 Business Objectives

### The dashboard enables businesses to:

- 📈 Monitor sales growth trends across different periods
- 👥 Analyze customer acquisition and repeat customer behavior
- 📦 Track inventory availability and damaged stock
- 💬 Understand customer feedback and sentiment trends
- 📢 Evaluate marketing campaign performance and ROI
- 🌍 Identify top-performing products and regions
- 🎯 Improve operational decision-making using interactive KPIs

---

# ⚙️ Solution Architecture

- 📂 Excel-based dataset integration
- ⚡ Power Query for ETL and data transformation
- 🧩 Star schema data modeling
- 🧠 Dynamic DAX calculations for KPI generation
- 📊 Interactive dashboard navigation using buttons & slicers
- 🔄 Dynamic filtering and cross-highlighting across visuals
